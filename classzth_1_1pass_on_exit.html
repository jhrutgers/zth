Detailed Description

Makes the fiber pass the given gate upon exit.

Example:

void foo() { ... }
zth_fiber(foo)
int main_fiber(int argc, char** argv) {
zth::Gate gate(3);
async foo() << zth::passOnExit(gate);
async foo() << zth::passOnExit(gate);
gate.wait();
// If we get here, both foo()s have finished.
return 0;
}
