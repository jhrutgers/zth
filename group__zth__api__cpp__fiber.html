◆ async

#define async

Run a function as a new fiber.

The function must have passed through zth_fiber() (or friends) first. Example:

void foo(int i) { ... }
zth_fiber(foo)
void main_fiber(int argc, char** argv) {
async foo(42);
}
Examples
1_helloworld.cpp, 2_fibers.cpp, 3_coop.cpp, 4_sync.cpp, 5_perf.cpp, daemon_pattern.cpp, fsm.cpp, measure.cpp, socks.cpp, and zmq.cpp.
